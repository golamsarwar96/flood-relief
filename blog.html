Frequently Asked Question (FAQ'S)

What is the Document Object Model (DOM)?

Document Object Model is a model which can be used to access HTML using JavaScript.It defines the logical structure of documents.
It DOM all the HTML element are considered nodes. The full HTML document is considered as a tree, where the root of the tree is html tag.
DOM is use to manipulate html elements. It can also style html document using different properties.

How do you select an element from the DOM?

Elements can be selected in multiple ways using DOM.


1. getElementsByTagName()

2. getElementById()

3. getElementByClassName()

4. querySelector()

5. querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener
to each and every similar element, we can add an event listener to a parent element and call an event
on a particular target using the .target property of the event object.

How do you manipulate an element's attributes and styles using the DOM? useful?

1. setAttribute() To change an element's attributes, you can use the setAttribute method or directly
manipulate the properties of the element.
example : element.setAttribute('title','this is a tooltip');


2. .style To change an element's styles, you can modify
the style property directly.
example : element.style.color = 'blue';
element.style.backgroundColor = 'yellow';
element.style.display = 'none';
